## Installation

1. Clone the repository:

```bash
$ git clone https://github.com/melNovaiss/productsControl.git
```

2. Install npm
```
npm install
```

3. Start server and backend
```
npm run serve
```
```
npm run backend
```

4. Open site
- Open your browser, then:
Acces `http://localhost:8080`
