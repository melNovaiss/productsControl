# Product Control
This repository contains a project built using Vue. 3, Bootstrap V5, json.server as the database.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

2. Install npm
